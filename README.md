# wx miniprogram starter

## Build in features

- TypeScript
- eslint
- UnoCss
- Vant Weapp

## Get started

1. Using template

```bash
npx degit guygubaby/wx-miniprogram-starter [local-name]
```

2. install deps

```bash
pnpm i
```

3. [Setup Vant](https://vant-ui.github.io/vant-weapp/#/quickstart)

4. Build npm using Wechat Devtools

5. Start dev server

```bash
pnpm dev
```
